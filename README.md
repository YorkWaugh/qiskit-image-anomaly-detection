# Qiskit Image Anomaly Detection

## Environment Requirements

The following environment has been tested and confirmed to work:

- Operating System: Windows 11
- Python Version: CPython 3.13.3

## Dependencies

The following versions have been tested and confirmed to work:

```plaintxt
matplotlib==3.10.3
numpy==2.2.6
pillow==11.2.1
qiskit==2.0.1
qiskit-aer==0.17.0
scipy==1.15.3
torch==2.7.0+cu128
torchvision==0.22.0+cu128
```

## How to Run

To start the project, simply run the following command in your terminal:

```cmd
python src/main.py
```

The dataset will be downloaded automatically if not present.
